## How to Use

### View Your Portfolio
Just visit: https://k36x7ryn92u5.space.minimax.io

### Interact with the Portfolio
1. **Navigate**: Click and drag to rotate camera, scroll to zoom
2. **Open Modals**: Click the navigation buttons (Work, About, Contact)
3. **Play Piano**: Click on the piano keys in the 3D scene
4. **Toggle Theme**: Click the sun/moon button for day/night mode
5. **Toggle Sound**: Click the speaker button to mute/unmute

1. Edit `index.html` (sections marked with clear headers)
2. Run `npm run build` to rebuild
3. Deploy the `dist/` folder
